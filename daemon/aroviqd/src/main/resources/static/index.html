AROVIQ Daemon (aroviqd)

DAEMON RUNNING

This is the local runtime daemon for AROVIQ BROWSE. It serves as the system source of truth, handling append-only memory, execution logic, and trust evaluation.

Available API Endpoints

  • /api/health/status - High-level health and uptime
  • /api/health/discovery - Registered system endpoints
  • /api/identity - Daemon name, version, and PID
  • /api/capabilities - Static system capabilities
  • /actuator/health - Standard Spring Actuator health
